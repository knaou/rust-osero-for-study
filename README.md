# Rust学習用プロジェクト：リバーシ（オセロ）

このプロジェクトは、Rustの初学者が実際のコードを通じてRustの基礎概念を学ぶためのサンプルプログラムです。
コンソール上で動作するシンプルなリバーシ（オセロ）ゲームを実装しています。

## 対象読者

- PythonやC言語などのプログラミング経験が多少ある方。
- Rustを学び始めたばかりで、基本的な構文や概念が実際のプロジェクトでどのように使われるかを知りたい方。

## 目的

- **Rust特有の概念の理解**: `mut`（可変性）、所有権・借用、列挙型（enum）、パターンマッチング（match）、トレイト（Displayなど）といったRustの核となる概念を、コメント付きのコードで学びます。
- **標準的なプロジェクト構造の学習**: Rustにおけるモジュールシステム（`mod`, `use`）や、`Cargo` を使ったパッケージ管理に慣れることができます。
- **実用的なロジックの実装**: 盤面管理や石を裏返す処理など、具体的なアルゴリズムがRustでどう記述されるかを確認できます。

## プロジェクト構造

```text
.
├── Cargo.toml      # プロジェクトの設定ファイル（依存関係など）
├── Makefile        # よく使うコマンドのショートカット
├── src
│   ├── main.rs     # エントリポイント（ゲームループと入出力）
│   └── game
│       ├── mod.rs    # モジュールの定義
│       ├── board.rs  # 盤面（Board構造体）のロジック
│       └── player.rs # プレイヤー（Player列挙型）の定義
└── LICENSE         # ライセンス情報
```

### ファイルの役割

- **`src/main.rs`**: ゲームの全体的な流れを制御します。ユーザーからの入力を受け取り、盤面を更新し、勝敗を判定します。
- **`src/game/player.rs`**: 黒（○）と白（●）のプレイヤーを定義します。Rustの `enum` の強力な機能や、表示のための `Display` トレイトの実装方法が学べます。
- **`src/game/board.rs`**: 8x8の盤面と、リバーシのルール（石を置けるかの判定、裏返し処理）を管理します。2次元配列の扱いや、`&self` と `&mut self` の使い分けが重要です。

## 使い方

### 準備

Rustがインストールされていることを確認してください。

### 実行方法

プロジェクトのルートディレクトリで以下のコマンドを実行します。

```bash
cargo run
```

または、`make` がインストールされている場合は以下も利用可能です。

```bash
make run
```

### 操作方法

1. 実行すると盤面が表示されます。
2. `Enter coordinates (row col), e.g., '3 2':` と表示されたら、石を置きたい場所の行と列をスペース区切りで入力します（例：3行目の2列目なら `3 2`）。
3. 交互に石を置き、両者置けなくなったらゲーム終了です。

## 学べる主なポイント

ソースコード内には、初心者向けに詳細な日本語コメントが付いています。特に以下の部分に注目してください。

- **可変性 (`mut`)**: なぜ必要なのか、どこで使うのか。
- **パターンマッチング (`match`, `if let`)**: 条件分岐を安全かつ強力に行う方法。
- **借用と所有権**: 関数にデータを渡す際の `&self` と `&mut self` の違い。
- **型変換**: `as` を使ったキャスト（`i32` から `usize` など）。
- **メソッドチェーン**: `split_whitespace().filter_map(...).collect()` といった現代的な記述。
